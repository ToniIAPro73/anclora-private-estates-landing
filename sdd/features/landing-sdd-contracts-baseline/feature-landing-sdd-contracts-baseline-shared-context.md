# Shared Context — Landing SDD Contracts Baseline

## Contexto

`sdd/contracts/` existe, pero aún no se ha poblado con contratos aplicables a esta landing.

Esta feature define el baseline contractual mínimo antes de continuar con integraciones funcionales.

## Clasificación

La landing pertenece al grupo `Ultra Premium` del ecosistema Anclora.

No debe evaluarse como:

- app interna
- app premium genérica
- portfolio showcase
- dashboard operativo

Debe evaluarse como landing pública ultra premium de `Anclora Private Estates`.

## Fuentes canónicas

### Fuente contractual

- Bóveda Anclora
- Repo privado: `git@github.com:ToniIAPro73/boveda-anclora.git`

### Fuente ejecutable

- `anclora-design-system`
- Repo: `https://github.com/ToniIAPro73/anclora-design-system.git`

## Regla operativa

La Bóveda gobierna contratos, decisiones y excepciones.

`anclora-design-system` materializa la implementación ejecutable mediante tokens, taxonomy, themes, foundations, components, patterns y assets.

## Restricción

No usar `docs/` local de la landing como fuente normativa en esta feature.
