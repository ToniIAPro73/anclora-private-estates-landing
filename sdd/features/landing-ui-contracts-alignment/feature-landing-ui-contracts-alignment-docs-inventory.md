# Docs Inventory — Landing UI Contracts Alignment

## Estado

Documento informativo no normativo.

## Decisión

La carpeta `docs/` queda fuera del alcance normativo de esta feature hasta nueva revisión.

No se usará como referencia para contratos SDD en este momento.

## Motivo

Los archivos de `docs/` pueden contener documentación histórica, material de trabajo, specs previas, fragmentos de implementación o referencias útiles, pero todavía no se ha validado si:

- cumplen con los contratos actuales de la Bóveda Anclora;
- son compatibles con `anclora-design-system`;
- pueden considerarse documentación genérica reutilizable;
- deben migrarse a `sdd/core/`, `sdd/contracts/`, `sdd/features/` o `_legacy`.

## Regla operativa

No mover, copiar, promover ni usar archivos de `docs/` como fuente normativa hasta que exista una feature específica de normalización documental.
