# Constitution Canonical — Anclora Private Estates Landing

## Estado

Documento base local para gobernanza del repo.

## Principios obligatorios

1. No introducir secretos en código, prompts, SDD ni commits.
2. No hardcodear URLs temporales de túnel en código fuente.
3. Toda integración externa debe ir mediante variables de entorno.
4. Toda captura de datos personales debe respetar consentimiento explícito.
5. Toda feature frontend debe declarar contratos UI afectados.
6. Toda feature debe tener SDD, test plan, QA report y gate final.
7. El código local no puede contradecir la Bóveda Anclora ni `anclora-design-system`.

## Fuentes superiores

- Bóveda Anclora.
- `anclora-design-system`.
