# Rules — Landing UI Contracts Alignment

## Scope

This feature is documentation-only.

## Allowed

- Audit UI contract documents.
- Compare legacy contracts with Bóveda Anclora and `anclora-design-system`.
- Classify contracts as `canonical`, `needs-update`, `legacy`, or `deprecated`.
- Update SDD documents and agent prompts for this feature.

## Forbidden

- Do not modify `src/`.
- Do not modify API, backend, webhook, n8n, or Nexus integration.
- Do not modify `.env` or secrets.
- Do not declare legacy UI contracts as canonical without explicit evidence.
- Do not commit or push without explicit approval.
