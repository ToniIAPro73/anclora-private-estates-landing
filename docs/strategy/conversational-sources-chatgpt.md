2. Tokens y reglas base
Color system

Base pública principal:

bg-canvas: #fbf9f4
bg-surface-low: crema cálido un tono por debajo
bg-surface-high: marfil más denso para cards elevadas
text-primary: casi negro con matiz verdoso
brand-primary: #003331
brand-primary-2: #004b49
brand-gold: #d4af37 solo para acentos puntuales o estados premium

Esto sale directamente del canon: fondo cálido #fbf9f4, teal profundo #003331 / #004b49, dark midnight #0a192f, oro muy medido y glass blur para menús flotantes.

Typography
display: Newsreader
body/ui: Manrope

No tocaría esta pareja. Está definida como el balance entre autoridad editorial y utilidad moderna.

Spacing rhythm

Usaría el ritmo no estándar del sistema:

micro padding: 0.7rem y 1rem
section breaks: 5.5rem y 7rem
overlap arquitectónico: 2.75rem

No es un detalle menor: ese ritmo es parte del acabado premium del prototipo.

Regla visual dura

No usar bordes de 1px como recurso principal. Separar por cambio de superficie, aire y contraste tonal. El propio sistema lo prohíbe casi explícitamente.

type CommonFieldProps = {
  label: string;
  name: string;
  value: string;
  onChange: (
    event: React.ChangeEvent<
      HTMLInputElement | HTMLTextAreaElement | HTMLSelectElement
    >,
  ) => void;
};

function Field({
  label,
  name,
  value,
  onChange,
  placeholder,
  type = "text",
}: CommonFieldProps & { placeholder?: string; type?: string }) {
  return (
    <label className="block">
      <span className="mb-2 block text-sm text-[var(--pe-text-soft)]">
        {label}
      </span>
      <input
        type={type}
        name={name}
        value={value}
        onChange={onChange}
        placeholder={placeholder}
        className="w-full rounded-2xl bg-[var(--pe-surface-high)] px-4 py-3 text-sm text-[var(--pe-text)] outline-none ring-0 placeholder:text-[var(--pe-text-muted)]"
      />
    </label>
  );
}

function SelectField({
  label,
  name,
  value,
  onChange,
  options,
}: CommonFieldProps & { options: string[] }) {
  return (
    <label className="block">
      <span className="mb-2 block text-sm text-[var(--pe-text-soft)]">
        {label}
      </span>
      <select
        name={name}
        value={value}
        onChange={onChange}
        className="w-full rounded-2xl bg-[var(--pe-surface-high)] px-4 py-3 text-sm text-[var(--pe-text)] outline-none"
      >
        <option value="">Selecciona una opción</option>
        {options.map((option) => (
          <option key={option} value={option}>
            {option}
          </option>
        ))}
      </select>
    </label>
  );
}

function TextAreaField({
  label,
  name,
  value,

  placeholder,
}: CommonFieldProps & { placeholder?: string }) {
